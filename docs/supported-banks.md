# Supported Banks

| Bank | Code | PDF Format | Status |
|------|------|-----------|--------|
| State Bank of India | `sbi` | Txn Date / Narration / Withdrawals / Deposits / Balance | ✅ Supported |
| HDFC Bank | `hdfc` | Date / Narration / Chq/Ref / Value Dt / Withdrawal / Deposit / Balance | ✅ Supported |
| ICICI Bank | `icici` | Date / Particulars / Withdrawal / Deposit / Balance | ✅ Supported |
| Axis Bank | `axis` | Date / Particulars / Debit / Credit / Balance | ✅ Supported |
