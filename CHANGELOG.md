## [0.1.0] - 2026-05-12

### Added
- Initial release
- SBI, HDFC, ICICI, Axis Bank PDF parser
- UPI narration parser with app/merchant/VPA extraction
- Auto bank detection from PDF content
- Transaction auto-categorization (20+ categories)
- Balance verification (row-by-row)
- Password-protected PDF support
- CLI tool (`bankparse`)
- Export to CSV, Excel (with summary), JSON
- Pandas DataFrame output with unified columns
